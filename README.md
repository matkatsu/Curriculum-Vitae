## 自己PR

### 設計から開発、運用まで経験

プロジェクト立ち上げ段階から取り組んだことも多く、どのようなフェーズにも対応できると自負しております。

1社目の「Dモーニング」「まんがライフGIGA」プロジェクトではエンジニア代表としてコンテンツ提供元の企業様と毎週MTGを行い企画会議に参加させていただきました。他のプロジェクトでもプロデューサー含めチーム全員で取り組むことができました。

2社目以降もフロントエンドからデータパイプラインに至るまで自身のスキルの幅を広げ、社内への技術共有・勉強会という形で還元してきました。

### 新しいモノ、知識、技術に対して積極的

目的を達成するために必要であれば、今まで経験したことのない新技術に触れることに抵抗は一切ありません。
現職ではサービスの一部でしか使用していなかったAngularJSを、フロントエンド全般の主要FWとして採用・開発することができました。AngularJSによる開発経験者はチーム内に自分以外居ない中、自身でも試行錯誤しつつ環境作りからチームに貢献することができました。
